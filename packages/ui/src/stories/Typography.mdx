# Typography

## How to use

[Figma reference](https://ant.design/components/typography/)

Use typography names as they are in Figma, but with some differences:

* Lowercase name and replace `/` with `-` in class names
* Replace `S` -> `sm`, `M` -> `md`, `L` -> `lg` in class names

Example: `TITLE/L/Bold` -> `.title-lg-bold`
